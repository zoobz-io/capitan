# Persistence

*Recipe: Build event storage and replay using observers and the replay API.*

Store events to a database for audit trails, debugging, and replay. An observer captures every event; a replay mechanism re-emits them on demand.

## The Scenario

Continuing the e-commerce service from [Distribution](3.distribution.md), we now want to persist every event for audit trails and debugging. When issues arise, we can replay historical events to reproduce problems or backfill new listeners.

## The Pattern

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│   Emit()    │────▶│   Capitan   │────▶│  Listeners  │
└─────────────┘     └──────┬──────┘     └─────────────┘
                          │
                    ┌─────▼─────┐
                    │  Observer │
                    │ (persist) │
                    └─────┬─────┘
                          │
                    ┌─────▼─────┐
                    │  Database │
                    └─────┬─────┘
                          │
                    ┌─────▼─────┐
                    │  Replay   │
                    └───────────┘
```

Events flow through capitan normally. An observer persists each event. Later, stored events can be replayed through the same signal infrastructure.

## Event Storage

Define a storage format that captures all event data:

```go
// storage/event.go
package storage

import (
    "time"

    "github.com/zoobzio/capitan"
)

// StoredEvent represents a persisted event.
type StoredEvent struct {
    ID        string         `json:"id"`
    Signal    string         `json:"signal"`
    Severity  string         `json:"severity"`
    Timestamp time.Time      `json:"timestamp"`
    Fields    map[string]any `json:"fields"`
}

// FromCapitanEvent converts a capitan event to storage format.
func FromCapitanEvent(e *capitan.Event) StoredEvent {
    fields := make(map[string]any)
    for _, f := range e.Fields() {
        fields[f.Key().Name()] = f.Value()
    }

    return StoredEvent{
        ID:        generateID(),
        Signal:    e.Signal().Name(),
        Severity:  string(e.Severity()),
        Timestamp: e.Timestamp(),
        Fields:    fields,
    }
}
```

## Persistence Observer

Attach an observer that writes every event to the database:

```go
// storage/observer.go
package storage

import (
    "context"
    "database/sql"
    "encoding/json"
    "fmt"

    "github.com/zoobzio/capitan"
)

type EventStore struct {
    db *sql.DB
}

func NewEventStore(db *sql.DB) *EventStore {
    return &EventStore{db: db}
}

// Observer returns a capitan observer that persists all events.
func (s *EventStore) Observer() capitan.EventCallback {
    return func(ctx context.Context, e *capitan.Event) {
        // Skip replay events to avoid duplicates
        if e.IsReplay() {
            return
        }

        stored := FromCapitanEvent(e)

        fieldsJSON, err := json.Marshal(stored.Fields)
        if err != nil {
            return // log error in production
        }

        _, err = s.db.ExecContext(ctx, `
            INSERT INTO events (id, signal, severity, timestamp, fields)
            VALUES ($1, $2, $3, $4, $5)
        `, stored.ID, stored.Signal, stored.Severity, stored.Timestamp, fieldsJSON)

        if err != nil {
            // Log but don't block - persistence is best-effort
            return
        }
    }
}

// Query retrieves events matching criteria.
func (s *EventStore) Query(ctx context.Context, opts QueryOpts) ([]StoredEvent, error) {
    query := `
        SELECT id, signal, severity, timestamp, fields
        FROM events
        WHERE 1=1
    `
    var args []any
    argN := 1

    if opts.Signal != "" {
        query += fmt.Sprintf(" AND signal = $%d", argN)
        args = append(args, opts.Signal)
        argN++
    }

    if !opts.Since.IsZero() {
        query += fmt.Sprintf(" AND timestamp >= $%d", argN)
        args = append(args, opts.Since)
        argN++
    }

    if !opts.Until.IsZero() {
        query += fmt.Sprintf(" AND timestamp <= $%d", argN)
        args = append(args, opts.Until)
        argN++
    }

    query += " ORDER BY timestamp ASC"

    if opts.Limit > 0 {
        query += fmt.Sprintf(" LIMIT $%d", argN)
        args = append(args, opts.Limit)
    }

    rows, err := s.db.QueryContext(ctx, query, args...)
    if err != nil {
        return nil, err
    }
    defer rows.Close()

    var events []StoredEvent
    for rows.Next() {
        var e StoredEvent
        var fieldsJSON []byte

        err := rows.Scan(&e.ID, &e.Signal, &e.Severity, &e.Timestamp, &fieldsJSON)
        if err != nil {
            return nil, err
        }

        json.Unmarshal(fieldsJSON, &e.Fields)
        events = append(events, e)
    }

    return events, rows.Err()
}

type QueryOpts struct {
    Signal string
    Since  time.Time
    Until  time.Time
    Limit  int
}
```

```sql
CREATE TABLE events (
    id TEXT PRIMARY KEY,

    severity TEXT NOT NULL,
    timestamp TIMESTAMPTZ NOT NULL,
```

## Wiring Persistence

Building on the e-commerce application, attach persistence alongside existing workflows:

```go
// main.go
package main

import (
    "context"

    "github.com/zoobzio/capitan"

    "myapp/inventory"
    "myapp/notifications"
    "myapp/orders"
    "myapp/payments"
    "myapp/storage"
)

func main() {
    ctx := context.Background()
    db := connectDB()
    store := storage.NewEventStore(db)

    // Persist all events
    observer := capitan.Observe(store.Observer())
    defer observer.Close()

    // Wire workflows (same as before)
    capitan.Hook(orders.OrderCreated, func(ctx context.Context, e *capitan.Event) {
        productID, _ := orders.ProductID.From(e)
        orderID, _ := orders.OrderID.From(e)
        quantity, _ := orders.Quantity.From(e)

        remaining := reserveStock(productID, quantity)

        capitan.Emit(ctx, inventory.StockReserved,
            inventory.ProductID.Field(productID),
            inventory.OrderID.Field(orderID),
            inventory.Quantity.Field(quantity),
            inventory.Remaining.Field(remaining),
        )

        if remaining < 10 {
            capitan.Warn(ctx, inventory.LowStock,
                inventory.ProductID.Field(productID),
                inventory.Remaining.Field(remaining),
            )
        }
    })

    capitan.Hook(payments.PaymentProcessed, func(ctx context.Context, e *capitan.Event) {
        orderID, _ := payments.OrderID.From(e)
        order := confirmOrder(orderID)

        capitan.Emit(ctx, orders.OrderConfirmed,
            orders.OrderID.Field(orderID),
        )

        // Skip notifications during replay
        if !e.IsReplay() {
            sendNotification(order.CustomerID, "email", "order_confirmed")
        }

        capitan.Emit(ctx, notifications.NotificationSent,
            notifications.CustomerID.Field(order.CustomerID),
            notifications.Channel.Field("email"),
            notifications.Template.Field("order_confirmed"),
        )
    })

    capitan.Hook(payments.PaymentFailed, func(ctx context.Context, e *capitan.Event) {
        orderID, _ := payments.OrderID.From(e)
        order := getOrder(orderID)

        releaseStock(order.ProductID, order.Quantity)

        capitan.Emit(ctx, inventory.StockReleased,
            inventory.ProductID.Field(order.ProductID),
            inventory.OrderID.Field(orderID),
            inventory.Quantity.Field(order.Quantity),
        )

        capitan.Emit(ctx, orders.OrderCanceled,
            orders.OrderID.Field(orderID),
        )

        // Skip notifications during replay
        if !e.IsReplay() {
            sendNotification(order.CustomerID, "email", "payment_failed")
        }

        capitan.Emit(ctx, notifications.NotificationSent,
            notifications.CustomerID.Field(order.CustomerID),
            notifications.Channel.Field("email"),
            notifications.Template.Field("payment_failed"),
        )
    })

    // Run application...

    capitan.Shutdown()
}
```

## Event Replay

Capitan provides `NewEvent` and `Replay` for re-emitting stored events:

```go
// storage/replay.go
package storage

import (
    "context"

    "github.com/zoobzio/capitan"
)

// SignalRegistry maps signal names to Signal values.
type SignalRegistry map[string]capitan.Signal

// KeyRegistry maps field names to typed keys for decoding.
type KeyRegistry map[string]func(any) capitan.Field

// Replay re-emits stored events through capitan.
func (s *EventStore) Replay(ctx context.Context, opts QueryOpts, signals SignalRegistry, keys KeyRegistry) error {
    events, err := s.Query(ctx, opts)
    if err != nil {
        return err
    }

    for _, stored := range events {
        signal, ok := signals[stored.Signal]
        if !ok {
            continue // Unknown signal, skip
        }

        fields := decodeFields(stored.Fields, keys)
        severity := parseSeverity(stored.Severity)

        // Construct event with original timestamp and severity
        e := capitan.NewEvent(signal, severity, stored.Timestamp, fields...)

        // Replay marks the event and processes it
        capitan.Replay(ctx, e)
    }

    return nil
}

func parseSeverity(s string) capitan.Severity {
    switch s {
    case "DEBUG":
        return capitan.SeverityDebug
    case "WARN":
        return capitan.SeverityWarn
    case "ERROR":
        return capitan.SeverityError
    default:
        return capitan.SeverityInfo
    }
}

func decodeFields(raw map[string]any, keys KeyRegistry) []capitan.Field {
    var fields []capitan.Field
    for name, value := range raw {
        if decoder, ok := keys[name]; ok {
            fields = append(fields, decoder(value))
        }
    }
    return fields
}
```

## Replay Use Cases

### Debugging a Failed Order

An order failed yesterday. Replay the events to trace what happened:

```go
func debugOrder(ctx context.Context, store *storage.EventStore, orderID string) {
    // Attach debug listener
    capitan.Observe(func(ctx context.Context, e *capitan.Event) {
        id, _ := orders.OrderID.From(e)
        if id == orderID {
            log.Printf("[%s] %s at %s: %v",
                e.Severity(),
                e.Signal().Name(),
                e.Timestamp().Format(time.RFC3339),
                extractFields(e),
            )
        }
    })

    yesterday := time.Now().AddDate(0, 0, -1)
    today := time.Now().Truncate(24 * time.Hour)

    store.Replay(ctx, storage.QueryOpts{
        Since: yesterday,
        Until: today,
    }, signalRegistry, keyRegistry)

    capitan.Shutdown()
}
```

Output:

### Backfilling Analytics

A new analytics listener was added. Replay historical events to populate it:

```go
func backfillAnalytics(ctx context.Context, store *storage.EventStore) {
    // New analytics listener
    capitan.Hook(orders.OrderCreated, func(ctx context.Context, e *capitan.Event) {
        total, _ := orders.Total.From(e)
        analytics.RecordOrder(e.Timestamp(), total)
    })

    capitan.Hook(payments.PaymentFailed, func(ctx context.Context, e *capitan.Event) {
        analytics.RecordFailure(e.Timestamp())
    })

    // Replay last 30 days
    store.Replay(ctx, storage.QueryOpts{
        Since: time.Now().AddDate(0, 0, -30),
    }, signalRegistry, keyRegistry)

    capitan.Shutdown()
}
```

## Considerations

### Replay Ordering

Events replay in timestamp order. Cross-signal ordering matches original emission order if timestamps are precise.

### Idempotency

Listeners should handle replay safely. Check `IsReplay()` for side effects:

```go
capitan.Hook(orders.OrderConfirmed, func(ctx context.Context, e *capitan.Event) {
    // Skip side effects during replay
    if !e.IsReplay() {
        sendConfirmationEmail(e)
        chargePayment(e)
    }
    // Always update analytics and internal state
    recordOrderConfirmed(e)
})
```

### Avoiding Duplicate Persistence

The persistence observer should skip replay events:

```go
func (s *EventStore) Observer() capitan.EventCallback {
    return func(ctx context.Context, e *capitan.Event) {
        if e.IsReplay() {
            return // Don't re-persist replayed events
        }
        // ... persist
    }
}
```

### Storage Growth

Events accumulate. Implement retention policies:

```go
func (s *EventStore) Prune(ctx context.Context, olderThan time.Duration) error {
    cutoff := time.Now().Add(-olderThan)
    _, err := s.db.ExecContext(ctx, `
        DELETE FROM events WHERE timestamp < $1
    `, cutoff)
    return err
}
```

### Selective Persistence

Persist only specific signals if storage is constrained:

```go
capitan.Observe(store.Observer(),
    orders.OrderCreated,
    orders.OrderConfirmed,
    orders.OrderCanceled,
    payments.PaymentFailed, // Persist failures for debugging
)
```

See [Concepts: Replay](../2.learn/2.concepts.md#replay) for API details on `NewEvent`, `Replay`, and `IsReplay`.
