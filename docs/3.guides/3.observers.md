# Observers Guide

Advanced patterns for using observers in capitan applications.

## What Are Observers?

Observers watch multiple signals without pre-registration. Unlike listeners that subscribe to specific signals, observers receive events from all signals (or a whitelist).

```go
// Listener: Specific signal
c.Hook(orderPlaced, handleOrder)

// Observer: All signals (or whitelist)
observer := c.Observe(auditHandler)
```

## Basic Observer Usage

### Observe All Signals

```go
// Create observer for all events
observer := c.Observe(func(ctx context.Context, e *capitan.Event) {
    log.Printf("[EVENT] %s at %s", e.Signal().Name(), e.Timestamp())
})
defer observer.Close()

// This observer receives ALL events
c.Emit(ctx, signal1, fields...)
c.Emit(ctx, signal2, fields...)
c.Emit(ctx, signal3, fields...)
```

### Observe Specific Signals (Whitelist)

```go
// Observer only receives whitelisted signals
sensitiveSignals := []capitan.Signal{
    userLogin,
    userLogout,
    passwordChanged,
    permissionGranted,
}

observer := c.Observe(securityAuditHandler, sensitiveSignals...)
defer observer.Close()

// Only these trigger the observer
c.Emit(ctx, userLogin, fields...)      // ✓ Observed
c.Emit(ctx, passwordChanged, fields...) // ✓ Observed

// These do NOT trigger the observer
c.Emit(ctx, userProfileUpdated, fields...) // ✗ Not observed
```

## Observer vs Listener

| Feature | Listener | Observer |
|---------|----------|----------|
| **Registration** | Per-signal (`Hook()`) | Dynamic (`Observe()`) |
| **Scope** | Single signal | All signals or whitelist |
| **Execution** | Async (worker queue) | Sync (direct call) |
| **Ordering** | Guaranteed per signal | No ordering guarantee |
| **Use Case** | Business logic | Cross-cutting concerns |
| **Performance** | O(1) per signal | O(n) observers on every emit |

```go
// Listener: Async, queued, ordered
c.Hook(orderPlaced, func(ctx context.Context, e *capitan.Event) {
    // Runs in signal worker goroutine
    // Queued with other orderPlaced events
    processOrder(e)
})

// Observer: Sync, direct, immediate
observer := c.Observe(func(ctx context.Context, e *capitan.Event) {
    // Runs in Emit() caller's goroutine
    // Blocks Emit() until complete
    auditLog(e)
})
```

## Observer Patterns

### Audit Logging

Track all events for compliance:

```go
type AuditLogger struct {
    db *sql.DB
}

func (al *AuditLogger) Start(c *capitan.Capitan) {
    observer := c.Observe(al.logEvent)
    // Keep observer alive for application lifetime
}

func (al *AuditLogger) logEvent(ctx context.Context, e *capitan.Event) {
    // Extract user from context
    userID, _ := ctx.Value("user_id").(string)

    // Save to database
    _, err := al.db.Exec(`
        INSERT INTO audit_log (timestamp, user_id, signal, severity, fields)
        VALUES (?, ?, ?, ?, ?)
    `, e.Timestamp(), userID, e.Signal().Name(), e.Severity(), fieldsToJSON(e.Fields()))

    if err != nil {
        log.Printf("Audit log error: %v", err)
    }
}
```

### Metrics Collection

Collect metrics on all events:

```go
type MetricsCollector struct {
    registry *prometheus.Registry
    counter  *prometheus.CounterVec
    latency  *prometheus.HistogramVec
}

func NewMetricsCollector() *MetricsCollector {
    counter := prometheus.NewCounterVec(
        prometheus.CounterOpts{
            Name: "capitan_events_total",
            Help: "Total number of events by signal",
        },
        []string{"signal", "severity"},
    )

    latency := prometheus.NewHistogramVec(
        prometheus.HistogramOpts{
            Name: "capitan_event_processing_seconds",
            Help: "Event processing latency",
        },
        []string{"signal"},
    )

    return &MetricsCollector{
        counter: counter,
        latency: latency,
    }
}

func (mc *MetricsCollector) Start(c *capitan.Capitan) {
    observer := c.Observe(mc.recordMetrics)
}

func (mc *MetricsCollector) recordMetrics(ctx context.Context, e *capitan.Event) {
    mc.counter.WithLabelValues(
        e.Signal().Name(),
        severityString(e.Severity()),
    ).Inc()

    // Track latency from timestamp to now
    latency := time.Since(e.Timestamp())
    mc.latency.WithLabelValues(e.Signal().Name()).Observe(latency.Seconds())
}
```

### Debug Tracing

Development-time event inspection:

```go
func StartDebugObserver(c *capitan.Capitan) *capitan.Observer {
    if os.Getenv("DEBUG_EVENTS") != "true" {
        return nil
    }

    observer := c.Observe(func(ctx context.Context, e *capitan.Event) {
        fmt.Printf("\n=== EVENT DEBUG ===\n")
        fmt.Printf("Signal:    %s\n", e.Signal().Name())
        fmt.Printf("Timestamp: %s\n", e.Timestamp().Format(time.RFC3339))
        fmt.Printf("Severity:  %d\n", e.Severity())
        fmt.Printf("Fields:\n")

        for _, field := range e.Fields() {
            fmt.Printf("  %s: %v\n", field.Key, field.Value)
        }

        // Print stack trace
        if os.Getenv("DEBUG_STACK") == "true" {
            fmt.Printf("\nStack:\n%s\n", debug.Stack())
        }

        fmt.Printf("===================\n\n")
    })

    return observer
}

// Usage
func main() {
    c := capitan.New()
    defer c.Shutdown()

    debugObserver := StartDebugObserver(c)
    if debugObserver != nil {
        defer debugObserver.Close()
    }

    // ... rest of application
}
```

### Event Replay/Recording

Record events for replay in tests:

```go
type EventRecorder struct {
    events []RecordedEvent
    mu     sync.Mutex
}

type RecordedEvent struct {
    Signal    string
    Timestamp time.Time
    Severity  capitan.Severity
    Fields    []capitan.Field
}

func (er *EventRecorder) Start(c *capitan.Capitan) *capitan.Observer {
    return c.Observe(er.recordEvent)
}

func (er *EventRecorder) recordEvent(ctx context.Context, e *capitan.Event) {
    er.mu.Lock()
    defer er.mu.Unlock()

    er.events = append(er.events, RecordedEvent{
        Signal:    e.Signal().Name(),
        Timestamp: e.Timestamp(),
        Severity:  e.Severity(),
        Fields:    e.Fields(), // Defensive copy
    })
}

func (er *EventRecorder) SaveToFile(path string) error {
    er.mu.Lock()
    defer er.mu.Unlock()

    data, err := json.MarshalIndent(er.events, "", "  ")
    if err != nil {
        return err
    }

    return os.WriteFile(path, data, 0644)
}

func (er *EventRecorder) Replay(c *capitan.Capitan, path string) error {
    data, err := os.ReadFile(path)
    if err != nil {
        return err
    }

    var events []RecordedEvent
    if err := json.Unmarshal(data, &events); err != nil {
        return err
    }

    for _, e := range events {
        signal := capitan.NewSignal(e.Signal, "Replayed signal")
        c.EmitWithSeverity(context.Background(), e.Severity, signal, e.Fields...)
    }

    return nil
}
```

### Security Monitoring

Monitor sensitive operations:

```go
type SecurityMonitor struct {
    alerter Alerter
}

func (sm *SecurityMonitor) Start(c *capitan.Capitan) {
    // Whitelist only security-relevant signals
    securitySignals := []capitan.Signal{
        userLogin,
        userLoginFailed,
        passwordChanged,
        permissionGranted,
        permissionDenied,
        dataExported,
        configChanged,
    }

    observer := c.Observe(sm.monitorEvent, securitySignals...)
}

func (sm *SecurityMonitor) monitorEvent(ctx context.Context, e *capitan.Event) {
    // Check for suspicious patterns
    if e.Signal() == userLoginFailed {
        sm.checkBruteForce(ctx, e)
    }

    if e.Signal() == permissionGranted {
        sm.checkUnauthorizedEscalation(ctx, e)
    }

    if e.Signal() == dataExported {
        sm.checkMassExport(ctx, e)
    }

    // Always log security events
    sm.logSecurityEvent(e)
}

func (sm *SecurityMonitor) checkBruteForce(ctx context.Context, e *capitan.Event) {
    userID := userIDKey.Extract(e)

    // Check failed login count in last 5 minutes
    count := sm.getRecentFailures(userID, 5*time.Minute)

    if count > 5 {
        sm.alerter.Send(Alert{
            Severity: "high",
            Message:  fmt.Sprintf("Potential brute force attack on user %s", userID),
            User:     userID,
        })
    }
}
```

### Rate Limiting Detection

Detect rate limit violations:

```go
type RateLimitMonitor struct {
    windows map[string]*Window
    mu      sync.RWMutex
}

type Window struct {
    count     int
    startTime time.Time
}

func (rlm *RateLimitMonitor) Start(c *capitan.Capitan) {
    // Monitor API-related signals
    apiSignals := []capitan.Signal{
        apiRequestReceived,
        apiRequestCompleted,
        apiRequestFailed,
    }

    observer := c.Observe(rlm.trackRequest, apiSignals...)
}

func (rlm *RateLimitMonitor) trackRequest(ctx context.Context, e *capitan.Event) {
    userID := userIDKey.Extract(e)
    endpoint := endpointKey.Extract(e)

    key := fmt.Sprintf("%s:%s", userID, endpoint)

    rlm.mu.Lock()
    defer rlm.mu.Unlock()

    window, ok := rlm.windows[key]
    if !ok || time.Since(window.startTime) > time.Minute {
        rlm.windows[key] = &Window{
            count:     1,
            startTime: time.Now(),
        }
        return
    }

    window.count++

    // Check threshold
    if window.count > 100 {
        c.EmitWithSeverity(ctx, capitan.SeverityWarn, rateLimitExceeded,
            userIDKey.Field(userID),
            endpointKey.Field(endpoint),
            countKey.Field(window.count),
        )
    }
}
```

### Event Transformation/Enrichment

Enrich events with additional data:

```go
type EventEnricher struct {
    userCache   *UserCache
    geoIP       *GeoIPLookup
}

func (ee *EventEnricher) Start(c *capitan.Capitan) {
    observer := c.Observe(ee.enrichEvent)
}

func (ee *EventEnricher) enrichEvent(ctx context.Context, e *capitan.Event) {
    // Extract user ID if present
    for _, field := range e.Fields() {
        if field.Key == "user_id" {
            userID := field.Value.(string)

            // Enrich context with user data
            user := ee.userCache.Get(userID)
            if user != nil {
                // Add to context for downstream handlers
                ctx = context.WithValue(ctx, "user", user)
            }
        }

        if field.Key == "ip_address" {
            ip := field.Value.(string)

            // Enrich with geo data
            location := ee.geoIP.Lookup(ip)
            if location != nil {
                ctx = context.WithValue(ctx, "location", location)
            }
        }
    }
}
```

## Observer Lifecycle Management

### Conditional Observers

Start/stop observers based on configuration:

```go
type ObserverManager struct {
    observers []*capitan.Observer
}

func (om *ObserverManager) Start(c *capitan.Capitan, config Config) {
    if config.EnableAudit {
        om.observers = append(om.observers,
            c.Observe(auditHandler),
        )
    }

    if config.EnableMetrics {
        om.observers = append(om.observers,
            c.Observe(metricsHandler),
        )
    }

    if config.EnableDebug {
        om.observers = append(om.observers,
            c.Observe(debugHandler),
        )
    }
}

func (om *ObserverManager) Stop() {
    for _, obs := range om.observers {
        obs.Close()
    }
    om.observers = nil
}
```

### Dynamic Observer Registration

Add observers at runtime:

```go
type DynamicObserverRegistry struct {
    c         *capitan.Capitan
    observers map[string]*capitan.Observer
    mu        sync.RWMutex
}

func (dor *DynamicObserverRegistry) Register(name string, handler capitan.EventCallback, signals ...capitan.Signal) {
    dor.mu.Lock()
    defer dor.mu.Unlock()

    // Close existing observer with same name
    if existing, ok := dor.observers[name]; ok {
        existing.Close()
    }

    // Register new observer
    observer := dor.c.Observe(handler, signals...)
    dor.observers[name] = observer
}

func (dor *DynamicObserverRegistry) Unregister(name string) {
    dor.mu.Lock()
    defer dor.mu.Unlock()

    if observer, ok := dor.observers[name]; ok {
        observer.Close()
        delete(dor.observers, name)
    }
}

// Usage: Enable/disable features at runtime
registry.Register("audit", auditHandler)
registry.Register("metrics", metricsHandler)

// Later: Disable metrics
registry.Unregister("metrics")
```

## Observer Performance

### Minimize Observer Count

Each observer adds O(n) overhead to every `Emit()`:

```go
// ❌ Bad: 10 observers = 10x overhead per emit
for i := 0; i < 10; i++ {
    c.Observe(func(ctx context.Context, e *capitan.Event) {
        // Handle event
    })
}

// ✅ Good: 1 observer, dispatch internally
type ObserverHub struct {
    handlers []capitan.EventCallback
}

func (oh *ObserverHub) Start(c *capitan.Capitan) {
    c.Observe(oh.dispatch)
}

func (oh *ObserverHub) dispatch(ctx context.Context, e *capitan.Event) {
    for _, handler := range oh.handlers {
        handler(ctx, e)
    }
}
```

### Use Whitelists

Reduce overhead by limiting observed signals:

```go
// ❌ Bad: Observer called for ALL events
observer := c.Observe(func(ctx context.Context, e *capitan.Event) {
    if e.Signal() == userLogin || e.Signal() == userLogout {
        // Handle
    }
    // Called for every event, even if not relevant
})

// ✅ Good: Observer only called for whitelist
observer := c.Observe(handler, userLogin, userLogout)
```

### Async Processing in Observers

Observers block `Emit()` - offload heavy work:

```go
type AsyncObserver struct {
    queue chan *capitan.Event
}

func (ao *AsyncObserver) Start(c *capitan.Capitan) {
    ao.queue = make(chan *capitan.Event, 100)

    // Background worker
    go ao.worker()

    // Observer just queues
    c.Observe(ao.enqueue)
}

func (ao *AsyncObserver) enqueue(ctx context.Context, e *capitan.Event) {
    // Copy event data (event will be pooled)
    copied := &capitan.Event{
        // Copy fields
    }

    select {
    case ao.queue <- copied:
    default:
        log.Println("Observer queue full, dropping event")
    }
}

func (ao *AsyncObserver) worker() {
    for e := range ao.queue {
        // Heavy processing here (doesn't block Emit)
        processEvent(e)
    }
}
```

## Observer Best Practices

### 1. Keep Observers Fast

Observers block `Emit()` - minimize work:

```go
// ✅ Good: Quick logging
c.Observe(func(ctx context.Context, e *capitan.Event) {
    log.Printf("[EVENT] %s", e.Signal().Name())
})

// ❌ Bad: Slow database write
c.Observe(func(ctx context.Context, e *capitan.Event) {
    db.Exec("INSERT INTO ...") // Blocks every Emit()!
})

// ✅ Better: Queue for async processing
c.Observe(func(ctx context.Context, e *capitan.Event) {
    queue <- e.Copy() // Quick enqueue, process async
})
```

### 2. Use Whitelists When Possible

```go
// Observe only relevant signals
observer := c.Observe(handler,
    userLogin,
    userLogout,
    passwordChanged,
)
```

### 3. Close Observers When Done

```go
observer := c.Observe(debugHandler)
defer observer.Close()
```

### 4. Handle Errors Gracefully

```go
c.Observe(func(ctx context.Context, e *capitan.Event) {
    if err := processEvent(e); err != nil {
        log.Printf("Observer error: %v", err)
        // Don't panic - would affect all emitters
    }
})
```

### 5. Avoid Side Effects in Observers

```go
// ❌ Bad: Modifying shared state
var counter int
c.Observe(func(ctx context.Context, e *capitan.Event) {
    counter++ // Race condition!
})

// ✅ Good: Use atomic operations
var counter atomic.Int64
c.Observe(func(ctx context.Context, e *capitan.Event) {
    counter.Add(1)
})
```

## Testing Observers

```go
func TestObserver(t *testing.T) {
    c := capitan.New()
    defer c.Shutdown()

    signal := capitan.NewSignal("test", "Test")

    // Capture observer calls
    var observed []string
    var mu sync.Mutex

    observer := c.Observe(func(ctx context.Context, e *capitan.Event) {
        mu.Lock()
        defer mu.Unlock()
        observed = append(observed, e.Signal().Name())
    })
    defer observer.Close()

    // Emit events
    c.Emit(context.Background(), signal)
    c.Shutdown()

    // Verify
    mu.Lock()
    defer mu.Unlock()
    assert.Equal(t, 1, len(observed))
}
```

## Production Observability

The patterns above demonstrate manual observability implementation. For production systems, consider [shotel](https://github.com/zoobzio/shotel) — an OpenTelemetry bridge for capitan that provides:

- **Automatic OTEL integration** — Events transform to logs, metrics, and traces
- **Type-safe field conversion** — Capitan fields become OTEL attributes
- **Span correlation** — Start/end signal pairs become distributed traces
- **Four metric types** — Counter, gauge, histogram, updowncounter

```go
import "github.com/zoobzio/shotel"

// Replace manual observers with shotel
cfg := &shotel.Config{
    Metrics: []shotel.MetricConfig{
        {Signal: apiRequestCompleted, Name: "api_requests_total", Type: shotel.MetricTypeCounter},
        {Signal: apiRequestCompleted, Name: "api_request_duration", Type: shotel.MetricTypeHistogram, ValueKey: latencyKey},
    },
    Traces: []shotel.TraceConfig{
        {Start: requestStart, End: requestEnd, CorrelationKey: &requestIDKey, SpanName: "http.request"},
    },
}

s, _ := shotel.New(c, logProvider, meterProvider, traceProvider, cfg)
defer s.Close()
```

Shotel is opinionated about capitan integration whilst remaining agnostic about OTEL provider configuration — you control exporters, sampling, and batching.
